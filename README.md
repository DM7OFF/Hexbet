# Hexbet - Modern Crypto Gambling Platform

A hybrid crypto gambling platform built with a modern stack, featuring both competitive esports-style PvP Ranked system and a classic fast casino mode.

## Architecture

* **Frontend:** React + TypeScript + Vite + TailwindCSS (Glassmorphism & Neon theme)
* **Backend:** Node.js + TypeScript + Express + Socket.io
* **Realtime:** WebSockets for PvP Matchmaking and live bets

## Features

1. **Ranked Mode (PvP)**
   * Skill-based matchmaking via ELO.
   * Real-time server-authoritative matches.
   * Players compete against each other for stakes.

2. **Normal Mode (Casino)**
   * Classic single-player games against the house (Crash, Dice, Plinko, Roulette).
   * Provably fair RNG implementation using HMAC-SHA256.

3. **Modern UI/UX**
   * Responsive layout with a high-end dashboard.
   * Fluid animations (Framer Motion) and custom neon accents.
   * Stitch-inspired premium design.

Open `http://localhost:5173` to view the application.

## Provably Fair System
The Casino mode guarantees fairness using a standard Server Seed, Client Seed, and Nonce cryptographic commitment scheme. 
* The server generates a secret seed and hashes it (SHA-256).
* The hash is shown to the user before the game starts.
* The game outcome is generated by combining the seeds via HMAC-SHA256.
* After the game, the server seed is revealed so the user can verify the exact outcome independently.

## PvP Matchmaking
Uses Socket.io to pool users into a queue. Matches are formed based on:
1. Game Type
2. Stake Amount
3. ELO Rank proximity
